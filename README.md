# TennisGame

  Exercice "Functional calisthenics" en langage Elixir

## Utilisation

### Installation de Elixir

https://elixir-lang.org/install.html

### Téléchargement des dépendances
 Dans le répertoire de travail :

``` mix deps.get ```

### Exécution des tests

```mix test ```

### Jouer

```mix run -e TennisGame.play```
